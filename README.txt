This is a mobile application for a game called Flag Your Food.

It uses Backendless as Baas and Sencha Touch Framework for the UI.

The structure of the tables required are described in backendless_init.js
Additionally, the image files are assumed to be in the following folders under the Project's files:
1) images of food - /dishes/
2) images of flags - /flags/

The background image (background.jpg) used in the logout page is in the folder
3) /media/
